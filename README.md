## 🌟 ¿Qué hace esta herramienta?
A diferencia de otros scripts genéricos, esta suite:
- **Auto-Aislamiento:** Detecta tus hilos de CPU y mueve la carga de la GPU y Red a los últimos núcleos, liberando el Core 0 para el sistema.
- **MSI Mode Inteligente:** Busca tu GPU NVIDIA y activa el modo MSI de forma segura.
- **Sin Hardcoding:** Ya no hay rutas ni IDs de hardware fijos. Funciona en una laptop o en una PC de escritorio de gama alta.
- **Visuales & Logs:** Interfaz "Matrix Style" con barra de progreso y consola de eventos en tiempo real.
