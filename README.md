# PaolozkyBot Optimizations 🚀

Scripts de optimización para Windows diseñados para maximizar el rendimiento en Gaming y Desarrollo.

## 📁 Contenido

- `scripts/turbo_gaming.ps1`: Optimiza el plan de energía, latencia de red y desactiva servicios innecesarios de DVR.

## 🚀 Cómo usar

1. Abre PowerShell como Administrador.
2. Clona este repositorio o descarga los archivos.
3. Ejecuta el script que necesites:
   ```powershell
   .\scripts\turbo_gaming.ps1
   ```
